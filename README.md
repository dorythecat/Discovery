# Discovery
A fun game I'm coding from the ground up on Vulkan :3

I want this to serve also as a basis for a more general vulkan game engine, taking from my original implementation for a vulkan game engine written from the ground up. There's a lot of stuff in store. I mainly want to have a basic rendering pipeline structured. Then, my idea is to have every other functionality, like movement, physics, simulation, animatio, model loading, and the like, interface with the main program through shared structures and functions, that would allow me to have a modular, modern, reliable, and highly optimized rendering backend for ALL of my project, both 2D and 3D! :D

As of now, I have only implemented basic rendering, without any of the cool features I'm planning on doing yet added. In its current state, it is just a highly optimized Bulkan renderer, though the final will be even more efficient, whicbh I'm still looking into. I wanna have a working version before going all frame-hunting! x3

# Note
This game is not even close to finished. Right now, you can't even move. So yeah... Not there yet xd
